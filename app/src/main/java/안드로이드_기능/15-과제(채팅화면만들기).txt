- RecyclerView 이용
- 채팅화면 구현

2단계

    - 검색을 해서 어떻게 구현할 수 있는지 찾아 본다.
    (검색하는 방법 + 검색 결과를 내 프로젝트에 적용하는 방법)
    -> 너무 많은 시간을 투자하지 마세요! (5시간 미만으로)
3단계
    - addview 를 이용해서 과제 2단게 구현
    - listview 를 이용해서 과제 2단계 구현
    -> 과제 풀이영상을 보고난 후에
